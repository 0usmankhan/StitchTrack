# **App Name**: StitchTrack POS

## Core Features:

- Order Input: Allows entry of orders with customer details, stitch types, materials, and delivery dates.
- Repair Tracking: Tracks repair orders separately, including descriptions of damages and repair instructions.
- Inventory Management: Keeps track of stock levels, helping manage fabrics and supplies.
- Automated Update Summaries: AI tool can generate a brief textual summary describing the order status, based on completion timestamps or order details.
- Reporting Dashboard: Offers reports and charts showing order volumes, repair trends, and stock needs.
- Customer Profiles: Stores customer history and preferences for repeat orders.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to inspire trust and professionalism.
- Background color: Light Gray (#F0F2F5), nearly desaturated Indigo, providing a clean backdrop.
- Accent color: Teal (#009688), a vibrant yet harmonious analog to Indigo, used for interactive elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, combining modern look and a little warmth/personality.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons representing order status, repair types, and inventory items.
- Subtle animations to indicate changes in order status and inventory updates.
